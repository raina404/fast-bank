# Fast Bank

A simple banking app built with Python and SQLite for my Code2College Elite 102 final project

## What it does
Fast Bank runs in the terminal and lets you manage bank accounts. You can create an account, put money in, take money out, and check your balance. Everything saves to a real database so your data doesn't disappear when you close the app.

## Features
- Create a new account with a starting deposit
- Deposit money
- Withdraw money (blocks you if you don't have enough)
- Check your balance
- See a list of all accounts
